SonicNote Sync 插件

将录音、转录、AI 总结同步到 Obsidian

将 SonicNote 妙记的录音、转录、AI 总结同步到 Obsidian，以 Markdown 文件管理。每条录音对应一个文件，包含完整的元数据和内容。

录音同步
一键拉取全部录音，自动生成 Markdown
AI 转录
含说话人识别和时间戳的逐字转录
智能总结
AI 自动生成的要点和行动项
自动同步
启动时自动同步 + 定时重同步

前置要求

  • Obsidian 桌面版 v0.15.0 及以上（下载地址）
  • SonicNote 妙记账号

安装

  1. 点击上方按钮下载 sonicnote-sync-v1.1.6.zip
  2. 解压 zip 文件，得到以下文件：
    • main.js
    • manifest.json
    • styles.css
  3. 打开 Obsidian Vault 所在文件夹，进入插件目录：
.obsidian/plugins/
  1. 在 plugins 目录下新建文件夹 sonicnote-sync，将解压的 3 个文件放进去：
.obsidian/plugins/sonicnote-sync/
├── main.js
├── manifest.json
└── styles.css
  1. 重启 Obsidian，进入 设置 → 第三方/社区插件，在列表中找到 SonicNote Sync 并启用

使用方法

登录

  1. 进入 设置 → SonicNote Sync
  2. 点击"登录"按钮
  3. 输入 API Key（在妙记 App → 我的 → API Key 管理中创建，格式：sk-xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx）

同步录音

登录成功后，通过以下任一方式触发同步：

  • 点击左侧功能区的 耳机图标
  • 命令面板（Ctrl/Cmd + P）输入 "SonicNote: 同步录音"

同步完成后，Vault 中会生成 SonicNoteSync/ 文件夹，每条录音对应一个 Markdown 文件。

同步内容

每个录音文件包含：

内容说明
Frontmatter录音元数据（ID、时间、时长、设备名、标签等）
笔记你在 App 中手写的笔记内容
AI 总结智能总结内容（要点、行动项等）
学习总结知识全景图、核心收获、课后巩固（如有）
转录内容AI 转写文本（含说话人、时间戳，可在设置中关闭）

设置项

设置默认值说明
同步文件夹SonicNoteSyncMarkdown 文件存放目录
包含转录内容开启关闭后同步文件中不包含逐字转录
启动时自动同步关闭打开 Obsidian 时自动执行一次同步
定时重同步关闭1/3/6/24 小时间隔自动重新同步
文件属性全部开启按需开关 Frontmatter 中的内置属性字段
自定义属性无添加自定义键值对到所有同步文件

卸载

  1. 进入 设置 → 社区插件，找到 SonicNote Sync，点击禁用并卸载
  2. 手动删除 .obsidian/plugins/sonicnote-sync/ 文件夹

AI Skills

从 Markdown 笔记中提取待办、生成工作总结

如何使用 Skill 安装提示词？

下面的文档不是直接使用的 Skill，而是"安装 Skill 的提示词"。需要把内容复制到支持本地 Skill / Agent 工作流的工具里使用，例如：

Codex · Claude Code · Claude 桌面版 · WorkBuddy · OpenClaw · OpenCode · Hermes Agent

使用方法：

  1. 打开支持创建本地 Skill 或 Agent 工作流的工具
  2. 新建一个对话或任务
  3. 打开提示词文档，全选复制全部内容
  4. 粘贴到工具的对话框里发送
  5. 按照工具提出的问题，填写你的个人配置（文件夹路径、写入位置、日记格式等）
  6. 工具会根据你的回答创建对应的 Skill 文件夹
  7. 创建完成后，按照工具给出的测试请求试跑一次

这些提示词不会直接处理你的文件。它们的作用是帮你创建一个可复用的 Skill。真正处理文件、提取待办或生成总结，是安装完成后的 Skill 来做。

Skill 仅 macOS

MD 待办提取器

从 Markdown 文件中智能提取待办事项，自动分类添加到 macOS 日历、提醒事项或备忘录。

下载安装提示词

智能分类规则

AI 自动判断每条待办的重要性和时间属性，分类到最合适的系统应用：

分类判断标准目标应用
📅 日历同半天内（≤4小时）有明确开始和结束时间的会议/讨论Calendar
⏰ 提醒事项跨度超半天，或只有截止时间无明确开始时间Reminders
📝 备忘录无明确时间但很重要Notes
⏭️ 跳过既无时间也不重要—

使用方法

基本用法

在支持 Skill 的 AI 工具中，告诉 AI 你要分析的文件或文件夹：

帮我分析 ~/Documents/projects 下的待办事项

工作流程

  1. 提供 md 文件路径或文件夹路径（可选指定时间范围）
  2. AI 扫描文件，识别潜在待办事项
  3. 显示分类结果表格，等待你确认
  4. 选择目标日历/提醒列表/备忘录文件夹
  5. 通过 AppleScript 自动添加到 macOS 系统应用

系统要求

  • 仅支持 macOS（依赖 AppleScript 操作日历、提醒事项、备忘录）
  • 首次使用需在 系统偏好设置 → 隐私与安全性 中授予日历、提醒事项、备忘录权限
Skill

工作总结生成器

从日记、会议记录、碎片笔记中生成多风格深度工作总结。核心理念：不要复述做了什么，要说出发现了什么。

下载安装提示词

13 种总结风格

根据日记内容智能推荐最佳风格，也可自选多种风格一次生成：

1. 模式发现型 2. 成果导向型 3. 风险预警型 4. 学习成长型 5. 叙事故事型 6. 极简清单型 7. OKR 对齐型 8. 时间审计型 9. 关系网络型 10. 决策复盘型 11. 情绪日志型 12. 跨界联想型 13. 古龙简洁型

风格说明

使用方法

基本用法

在支持 Skill 的 AI 工具中，告诉 AI 你想总结哪段时间：

帮我总结今天的工作

工作流程

  1. 获取素材 — 确认时间范围，自动查找日记文件
  2. 选择风格 — 展示全部 13 种风格，智能推荐 + 自选
  3. 生成总结 — 按选定风格的 prompt 模板深度生成
  4. 输出方式 — 对话展示 / 追加到日记 / 生成独立文件

Obsidian 集成

如果 Obsidian 正在运行且支持 CLI，Skill 会优先通过 CLI 操作文件（读取、追加、创建、搜索），无需手动指定完整路径。
